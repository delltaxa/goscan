# goscan

Port scanner written in golang

# Installation

Install and direct

```bash
git clone https://github.com/delltaxa/goscan
cd goscan/
```

# Build

```bash
go build .
```

# Usage 

```bash
./goscan tcp://127.0.0.1:1-1000
```
